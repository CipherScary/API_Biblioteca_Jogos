# Biblioteca de Jogos 

Feito por Sebastião Marques Silva Junior e Pedro Lucas Ponciano Silva

Projeto CRUD desenvolvido em Java utilizando:

- Java Servlet
- JSP
- Maven
- MySQL
- MVC
- DAO
- Service

## Funcionalidades

- Cadastro de jogos
- Listagem
- Atualização
- Exclusão

## Tecnologias

- Java
- JSP
- Servlet
- MySQL
- HTML
- CSS

## Como executar

1. Criar banco MySQL
2. Configurar ConnectionFactory
3. Rodar Maven
4. Deploy no Tomcat
